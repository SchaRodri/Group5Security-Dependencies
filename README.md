## Reusable Security Template

This repository can be used as a template for secure GitHub projects.

### Included features:
- Dependabot (automatic dependency updates)
- SBOM generation (CycloneDX)
- Security policy (SECURITY.md)
- Code ownership (CODEOWNERS)
- Branch protection rules (configured)

### How to use:
1. Copy the `.github` folder into your project
2. Adjust dependabot.yml if needed
3. Enable security features in GitHub settings

This ensures better security and governance for any project.
